# 🚗 Autonomous Line Follower V1

An autonomous line-following vehicle developed from a modified Formula 1 RC chassis using Arduino Uno. The project integrates embedded programming, electronics and mechanical adaptation to create a functional prototype capable of autonomous navigation and Bluetooth manual control.

---

## 📖 Project Overview

This project was developed as a personal engineering challenge to improve my knowledge of embedded systems, electronics and programming while preparing my application for the ARUS Formula Student team at the University of Seville.

The vehicle follows a black line using five infrared sensors and automatically corrects its steering according to the detected position of the line. It can also be manually controlled from a smartphone through Bluetooth.

---

## ✨ Main Features

- Autonomous line following
- Bluetooth manual control
- Dual operating modes
- Five infrared sensors
- Real-time steering correction
- Adjustable speed according to trajectory
- Modular Arduino code
- Telemetry through Serial Monitor

---

## 🛠 Hardware

- Arduino Uno
- HC-06 Bluetooth Module
- L298N Motor Driver
- 5 Infrared Sensors
- Rear DC Motor
- Front Steering Motor
- RC Formula 1 Chassis
